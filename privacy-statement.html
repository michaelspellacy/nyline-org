We have created this privacy statement in order to demonstrate our firm commitment to privacy. The following discloses the information gathering and dissemination practices for nyline.org

Information Automatically Logged

Each time a visitor comes to this Web site, our host servers collect some basic technical information, including, but not limited to the visitor's domain name, IP address, and geographic location. This data is compiled automatically via web server logs, but is not shared with others.

Personal Information

This Web site may request information from you to use certain features of the site, such as mailing list, membership sign up, and forums. We do not share any personal information with any third parties nor do we use any personal information for any other purposes.

Statistic Tracking

This website uses Google Analytics to analyze traffic to this web site. Google Analytics does not create individual profiles for visitors. Google Analytics only collects aggregate data.

External Links

This site contains links to other sites. We are not responsible for the privacy practices or the content of any website outside of nyline.org or nylineboards.org.

Public Forums

This site makes forums, message boards, and/or news groups available to its users. Please remember that any information that is disclosed in these areas becomes public information and you should exercise caution when deciding to disclose your personal information.
